## Getting Started

**This is a simple calculator built with react components.**

To get a local copy up and running follow these simple example steps.

### Prerequisites

    - npm
    - yarn
    - node

### Setup

    - install the dependencies from above.

### Install

Run

    - sudo yarn install

### Usage

    - yarn start

### Run tests

    - yarn test
